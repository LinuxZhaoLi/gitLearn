svn和git的区别：

svn:是增量式的管理方式。 版本控制在服务器， 服务器宕机。历史版本数据丢失。 

git: .GIT是分布式的. 本地有版本管理， 避免单点故障。 



本地库：存历史版本  git commit

暂存区： 临时存储  git add 

工作区： 写代码



代码托管中心：

局域网： gitLab

外网： github, 码云。

本地库和远程库：

本地到远程库： push 

远程库下载到本地： clone

本地更新： pull

克隆远程库：fork
